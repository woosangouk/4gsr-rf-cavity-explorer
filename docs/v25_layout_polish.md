# v25 Layout Polish

## Changes
1. Parameter group cards
- Left input group width is fixed.
- Group cards have fixed collapsed and expanded heights.
- Expanded content scrolls inside the group, so the sidebar structure stays stable.

2. Overview label layout
- Cavity image reduced slightly.
- Annotation labels repositioned to avoid overlapping the cavity center.

3. KPI and bottom summary widgets
- Units are placed inline after the numbers.
- Values and units are centered vertically and horizontally.
- Font sizes are clamped to avoid overflow.

4. Footer version
- Footer now shows `Version V25`.
