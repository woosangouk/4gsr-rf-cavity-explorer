# v20 Refined Parameter Search

## Change
The previous search box visually broke the GUI layout.
v20 replaces it with a compact pill-style parameter search that matches the existing white/blue dashboard theme.

## Behavior
- Type parameter or result name.
- Press Enter or click the round arrow button.
- The GUI switches to the relevant tab and highlights the target.
- Ctrl+K focuses the search box.

## Visual design
- Rounded command-palette style search
- Small embedded icon
- Compact round submit button
- Glass-like dropdown suggestions
