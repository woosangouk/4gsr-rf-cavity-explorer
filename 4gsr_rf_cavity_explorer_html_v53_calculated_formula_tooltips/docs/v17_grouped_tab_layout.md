# v17 Grouped Tab Layout

## Change
The previous layout had widgets scattered across the page. v17 reorganizes the interface:

- The top tabs now switch the main view panel.
- The right-side `Calculated Results` rail remains fixed/sticky.
- Seven parameter widgets remain below the main view as a stable summary.
- The left input panel remains fixed.
- The 3D View is integrated with labels in the Overview panel.

## Main screen grouping

Tabs:
- Overview
- Beam & Ring
- Cavity
- Coupling & Q
- Power
- Detune
- HPRF & Margin
- Diagnostics

Fixed elements:
- Left Input Parameters
- Right Calculated Results
- Bottom seven summary widgets
