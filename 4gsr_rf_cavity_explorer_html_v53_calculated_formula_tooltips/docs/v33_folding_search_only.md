# v33 Folding / Search Only Fix

## Scope
Only the parameter folding and search-triggered folding behavior was changed.

## Fixes
- Clicking anywhere on a blue parameter row opens/closes its drawer.
- Drawer is positioned relative to the sidebar, not the clicked row's text.
- Searching a term opens the necessary parameter group drawer.
- Example: searching `tuner` opens the Tuner Mapping-related drawer.
- Existing KPI / layout / search visual style was not intentionally changed.
