# v49 Detune Semicircle Gauge

## Scope
Only the Detune Estimate gauge was changed.

## Changes
- Rebuilt gauge as a reference-style semicircle meter.
- Blue-to-red semicircle arc from −90° to +90°.
- Major labels: −90°, −60°, −30°, 0°, +30°, +60°, +90°.
- Needle rotates around the bottom pivot.
- Digital value box shows current detune angle.
- Large lower duplicate detune number is removed.
