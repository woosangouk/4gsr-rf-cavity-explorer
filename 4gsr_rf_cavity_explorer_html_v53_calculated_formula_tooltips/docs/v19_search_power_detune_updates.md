# v19 Search / Power Flow / Detune Updates

## Search
A top search box was added.
It can find input parameters, result values, or view panels.
When a parameter is found, the screen moves to the relevant tab and highlights the target.

## Graphs
Cavity Count Sweep and Beam Current Sweep now show:
- Point values
- X-axis title
- Y-axis title

## Power
RF Power Flow was expanded and centered:
SSPA → Line Loss → Coupler → Cavity → Beam

Dissipated and Reflected are shown below with tighter, balanced spacing.

## Detune
The gauge now shows:
- Left side: negative angle
- Center: 0°
- Right side: positive angle

The color gradient means:
- green: small detune
- yellow/orange: transition
- red: large detune warning range

The tuner plunger tuning range reference graph is inserted below the Detune panel.
