# v36 Inline-only Accordion

## Fix
All previous side drawer/fixed-position folding behavior was removed.

## Behavior
- Click a blue parameter group.
- It opens directly below that group.
- Lower groups move downward.
- Clicking it again closes it and lower groups move upward.
- Search opens only the necessary group in the same inline way.

## Scope
Only parameter folding behavior was changed.
