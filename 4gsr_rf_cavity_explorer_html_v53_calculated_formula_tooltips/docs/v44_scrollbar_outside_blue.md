# v44 Scrollbar Outside Blue Sidebar Area

## Scope
Only the left Input Parameters scrollbar placement was changed.

## Fix
- The dark blue sidebar has a fixed width and ends before the scrollbar.
- The scrollbar is drawn in a separate light gutter outside the blue area.
- Blue cards keep their exact width when folders open/close.
- The app grid reserves the gutter width so the main dashboard does not overlap the external scrollbar.
