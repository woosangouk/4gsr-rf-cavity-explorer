# v35 Inline Accordion Folding

## Change
Parameter groups no longer open as a side drawer.

## Behavior
- Click a blue parameter row.
- Its setting fields open directly below that row.
- Lower rows move downward while the group is open.
- Click the same row again or another row to close/move the opened group.
- Search opens only the required parameter group inline.

## Scope
Only the parameter folding/opening behavior was changed.
Other UI sections were intentionally left unchanged.
