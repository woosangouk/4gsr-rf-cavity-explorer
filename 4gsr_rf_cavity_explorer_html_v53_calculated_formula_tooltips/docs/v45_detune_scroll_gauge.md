# v45 Detune Scroll / Gauge Update

## Scope
Only the Detune tab/widget was changed.

## Changes
- Detune tab now has internal vertical scrolling when content is larger than the widget.
- Tuner Plunger Tuning Range Reference can be viewed by scrolling inside the Detune panel.
- The 0° label is moved to the upper center near the needle-tip reference.
- The current detune angle label is drawn near the needle tip and placed to the right of the needle to avoid overlap.
