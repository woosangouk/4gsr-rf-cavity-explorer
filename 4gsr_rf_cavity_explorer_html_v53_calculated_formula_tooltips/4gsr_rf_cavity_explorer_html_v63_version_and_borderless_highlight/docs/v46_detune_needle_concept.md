# v46 Detune Needle Concept

## Scope
Only the Detune Estimate gauge was changed.

## Changes
- Replaced the old color-arc gauge with the user's proposed needle concept.
- Center top `0°` reference is shown with a vertical dashed line.
- Horizontal dashed reference line separates negative and positive angle direction.
- `- Angle°` is placed on the left and `+ Angle°` on the right.
- Current detune value is dynamically placed near the needle tip:
  - negative angle: left side of the needle
  - positive angle: right side of the needle
- The existing Detune panel internal scroll from v45 is preserved.
