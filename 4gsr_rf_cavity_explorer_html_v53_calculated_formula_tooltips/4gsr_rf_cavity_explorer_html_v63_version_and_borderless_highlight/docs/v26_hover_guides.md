# v26 Hover Guides

## Added
- Top navigation tabs show a guide tooltip after 1 second hover.
- Collapsed parameter groups show a short summary of the internal representative settings after 1 second hover.

## Behavior
- Hover over Overview ~ Effect Map tabs for 1 second.
- Hover over a collapsed parameter group for 1 second.
- The tooltip disappears when the mouse leaves.

## Purpose
This keeps the UI compact while still explaining what each tab or collapsed parameter block contains.
